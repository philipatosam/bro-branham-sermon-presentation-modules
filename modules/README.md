## How Sermons Are Structured

In every module, sermons follow a Bible-like structure so presentation software can handle them natively:

| Sermon data | Bible equivalent |
|---|---|
| Sermon collection | Bible |
| Each sermon | Book (e.g. `47-0412 · Faith Is the Substance`) |
| Each sermon | 1 Chapter |
| Each paragraph | Verse (paragraph number = verse number) |

This means you can navigate to any sermon, jump to a specific paragraph by number, and display one paragraph per slide or by your preference.
